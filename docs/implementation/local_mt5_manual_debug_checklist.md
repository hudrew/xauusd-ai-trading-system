# 本地 MT5 手工联调 Checklist

## 使用方式

每次手工联调时，从上到下勾选。

建议先在 demo 环境完成，再考虑 live。

---

## A. 基础信息

- [ ] 调试日期已记录
- [ ] 调试账户类型已确认：`demo / live`
- [ ] 券商服务器名已确认
- [ ] MT5 终端版本已确认
- [ ] 当前使用品种名已确认

建议记录：

- 服务器名：
- 账户号：
- 品种名：
- 终端路径：

---

## B. 终端状态

- [ ] MT5 已正常登录
- [ ] 连接状态正常
- [ ] `XAUUSD` 或实际券商品种已加入 `Market Watch`
- [ ] `M1` 图表可正常打开
- [ ] 图表有连续更新的 K 线
- [ ] bid / ask 在跳动

---

## C. 品种与配置对齐

- [ ] 券商实际品种名已确认是否就是 `XAUUSD`
- [ ] 点位精度已确认
- [ ] 最小下单手数已确认
- [ ] 允许的止损止盈最小距离已确认
- [ ] 交易时段限制已确认

项目配置映射：

- `XAUUSD_AI_MT5_SERVER`
- `XAUUSD_AI_MT5_PATH`
- `XAUUSD_AI_MT5_SYMBOL`
- `execution.mt5.deviation`
- `risk.max_spread_ratio`

---

## D. 行情与点差

- [ ] 常规时段点差范围已观察
- [ ] 欧盘 / 美盘活跃时段点差已观察
- [ ] 新闻前后点差扩张情况已观察
- [ ] 是否存在长时间卡价或跳价异常已观察

建议记录：

- 正常点差范围：
- 活跃时段点差范围：
- 新闻时段点差范围：
- 是否有明显异常：

---

## E. 交易权限

- [ ] 账户允许交易
- [ ] 日志未出现 `trading has been disabled - disabled on server`
- [ ] 终端未被禁止下单
- [ ] 自动交易相关开关状态已确认
- [ ] demo 环境最小手工单允许成交

---

## F. 手工小单验证

- [ ] 最小手数开仓成功
- [ ] 平仓成功
- [ ] 止损可设置
- [ ] 止盈可设置
- [ ] 手续费表现已观察
- [ ] 滑点表现已观察

建议记录：

- 开仓时间：
- 下单方向：
- 手数：
- 成交是否顺利：
- 是否有拒单：
- 手续费 / 滑点备注：

---

## G. 项目侧联调

- [ ] `smoke` 已运行
- [ ] 输出的状态 / 信号 / 风控结果已查看
- [ ] 历史回放已运行（如果已有 CSV）
- [ ] 高波动预警输出已查看

命令：

```bash
PYTHONPATH=src ./.venv/bin/python -m xauusd_ai_system.cli smoke
PYTHONPATH=src ./.venv/bin/python examples/replay_csv.py /path/to/xauusd_m1.csv
```

---

## H. 是否可以进入下一阶段

只有全部满足再进入执行宿主机阶段：

- [ ] 终端登录稳定
- [ ] 品种名和配置字段已对齐
- [ ] 手工最小单没有权限问题
- [ ] 点差和交易时段基本清楚
- [ ] 当前没有明显券商端限制
- [ ] 当前账户没有被服务器端禁用交易

如果以上还有未确认项，先不要切到 `paper/prod` 执行宿主机联调。
